# Project Overview: Soteria

## Purpose
Soteria is a professional environmental monitoring platform. It integrates NASA satellite imagery (GIBS/EONET), Open-Meteo telemetry, and Google Gemini AI analytics to provide real-time environmental analysis.

## Key Features
- **Live Satellite Archive**: Daily images from NASA Terra (MODIS) with a time navigator (2020-2026).
- **AI Environmental Reports**: Automated reports and recommendations generated by Gemini based on marine and air data.
- **Environmental Metrics**: AQI (PM2.5, NO2, O3, CO), surface temperature, wave height, and currents.
- **Water Level Tracker**: Visualization of historical trends in water level changes.
- **Green Energy Potential**: Wind and solar energy modeling based on Betz's law.

## Codebase Structure
- `app/`: Pages and API routes.
- `components/`: Reusable React components.
- `lib/`: Data source integration logic (Gemini, NASA, Open-Meteo).
- `context/`: React context providers.
- `public/`: Static assets.
- `docs/`: Documentation for APIs and visualizations.
